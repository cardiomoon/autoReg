# Add model summary to an object of class gaze

Add model summary to an object of class gaze

## Usage

``` r
addFitSummary(df, fit, statsname = "")
```

## Arguments

- df:

  An object of class "gaze" or "autoReg"

- fit:

  An object of class "glm" or "lm" or "crr"

- statsname:

  character Name of statistics

## Value

addFitSummary returns an object of
[`gaze`](https://cardiomoon.github.io/autoReg/reference/gaze.md) or
[`autoReg`](https://cardiomoon.github.io/autoReg/reference/autoReg.md) -
the same class as df

## Examples

require(dplyr)
#> Loading required package: dplyr
