# Highlight a data.frame

Highlight a data.frame

## Usage

``` r
highlight2(x, i = NULL, j = NULL, style = NULL, include.colname = FALSE)
```

## Arguments

- x:

  A data.frame

- i:

  numeric rows to highlight

- j:

  numeric columns to hightlight

- style:

  A style function or NULL

- include.colname:

  logical Whether or not include colname

## Value

a data.frame
