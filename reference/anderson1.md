# Remission survival times of 42 leukemia patients

A dataset containing survival time of 42 leukemia patients This data is
the same data with anderson, but sex and rx variable are factors not
numeric

## Usage

``` r
anderson1
```

## Format

A data.frame with 42 rows and 5 variables

- time:

  survival time in weeks

- status:

  censoring status 1=failure 0=censored

- sex:

  sex "Female" or "Male

- logWBC:

  log white blood cell count

- rx:

  treatment status "treatment" or "control"

## Source

David G. Kleinbaum and Mitchel Klein. Survival Analysis. A Self-Learning
Text(3rd ed)(Springer,2012) ISBN: 978-1441966452
